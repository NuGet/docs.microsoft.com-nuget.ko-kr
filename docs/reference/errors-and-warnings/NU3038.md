# <a name="nuget-warning-nu3038"></a>NuGet 경고 NU3038

### <a name="issue"></a>문제

인증 설정을 리포지토리 연대 필요 없는 패키지 리포지토리 연대 서명을 합니다.


### <a name="solution"></a>솔루션

연대 리포지토리 패키지가 포함 된 패키지 소스를 사용 합니다.
