# <a name="nuget-warning-nu3035"></a>NuGet 경고 NU3035

### <a name="issue"></a>문제

리포지토리 연대 서명 인증서에 대 한 완전 한 인증서 체인을 빌드할 수 없습니다.


### <a name="solution"></a>솔루션

문제의 원인을 검색 하는 타임 스탬프 인증 기관에 문의 합니다.
