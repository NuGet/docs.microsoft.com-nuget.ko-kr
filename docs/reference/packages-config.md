## <a name="examples"></a><span data-ttu-id="65799-141">예</span><span class="sxs-lookup"><span data-stu-id="65799-141">Examples</span></span>

<span data-ttu-id="65799-142">다음과 같은 `packages.config`은 두 개의 종속성을 가리킵니다.</span><span class="sxs-lookup"><span data-stu-id="65799-142">The following `packages.config` refers to two dependencies:</span></span>

```xml

  <package id="jQuery" version="3.1.1" targetFramework="net46" />
  <package id="NLog" version="4.3.10" targetFramework="net46" />
</packages>
```

<span data-ttu-id="65799-143">다음과 같은 `packages.config`는 9개의 패키지를 참조하지만 `developmentDependency` 특성 때문에 사용 중인 패키지를 빌드할 때 `Microsoft.Net.Compilers`가 포함되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="65799-143">The following `packages.config` refers to nine packages, but `Microsoft.Net.Compilers` will not be included when building the consuming package because of the `developmentDependency` attribute.</span></span> <span data-ttu-id="65799-144">또한 Newtonsoft.Json에 대한 참조는 8.x 및 9.x 버전에 대한 업데이트를 제한합니다.</span><span class="sxs-lookup"><span data-stu-id="65799-144">The reference to Newtonsoft.Json also restricts updates to 8.x and 9.x versions only.</span></span>

  <package id="Microsoft.Web.Xdt" version="2.1.1" targetFramework="net46" />
  <package id="Newtonsoft.Json" version="8.0.3" allowedVersions="[8,10)" targetFramework="net46" />
  <package id="NuGet.Core" version="2.11.1" targetFramework="net46" />
