        <h1>NuGet 설명서</h1>
        <p>NuGet은 .NET의 패키지 관리자입니다. 개발자는 NuGet으로 유용한 .NET 라이브러리를 만들고 공유하고 사용할 수 있습니다. NuGet 클라이언트 도구는 이러한 라이브러리를 “패키지”로 생성하고 사용할 수 있는 기능을 제공합니다.</p>
